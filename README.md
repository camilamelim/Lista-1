Lista de Exercícios em C (Fundamentos de Programação)

## 📌 Descrição

Este repositório contém a implementação, em linguagem C, de uma lista de exercícios.

O objetivo principal foi consolidar os **fundamentos da programação estruturada**, com foco em lógica, modelagem de algoritmos e resolução de problemas computacionais.

## 🎯 Competências Desenvolvidas

Durante a resolução dos exercícios, foram aplicadas e desenvolvidas as seguintes habilidades técnicas:

* Estruturação de algoritmos sequenciais
* Manipulação de entrada e saída padrão (`stdio.h`)
* Uso de variáveis e tipos primitivos (`int`, `float`)
* Aplicação de operadores:

  * Aritméticos (`+`, `-`, `*`, `/`, `%`)
  * Relacionais (`>`, `<`, `==`, `!=`)
* Controle de fluxo com estruturas condicionais (`if`, `else if`, `else`)
* Implementação de regras de negócio baseadas em lógica condicional
* Interpretação de problemas e tradução para código

## 🧩 Escopo dos Exercícios

A lista é composta por **17 problemas**, cobrindo diferentes cenários computacionais, como:

* Validação de condições (ex: aprovação por média)
* Comparação e ordenação básica de valores
* Operações matemáticas e modulares
* Conversões de unidades (temperatura, tempo, peso)
* Cálculos financeiros:

  * Juros simples
  * Percentuais
  * Reajustes salariais
  * Impostos
* Simulação de regras do mundo real (salário com hora extra, consumo de energia, investimentos)

## ⚙️ Abordagem Técnica

Cada exercício foi implementado de forma independente, priorizando:

* Clareza e legibilidade do código
* Separação lógica por problema
* Uso direto de estruturas básicas da linguagem (sem abstrações avançadas)
* Foco em fundamentos (base para estruturas futuras como loops, funções e vetores)

## 🛠️ Tecnologias e Ferramentas

* Linguagem: **C**
* Ambiente de desenvolvimento: **Visual Studio Code**

## 📈 Evolução Técnica

Este projeto representa o início da minha formação em programação, com foco em:

* Construção de base sólida em lógica computacional
* Preparação para tópicos mais avançados:

  * Estruturas de repetição
  * Funções
  * Vetores e matrizes
  * Estruturas de dados

## 💡 Considerações

Embora os exercícios sejam introdutórios, eles demonstram domínio dos conceitos fundamentais exigidos para evolução em desenvolvimento de software, especialmente em linguagens de baixo nível como C.
